# 象棋学习计划与提高要点

## 一、学习阶段规划

### 第一阶段：入门基础（1-2周）
- [ ] 熟悉棋盘：九宫、河界、交叉点
- [ ] 掌握七种棋子的走法与吃子规则
  - 车：横竖直线任意格
  - 马：走"日"字，注意蹩马腿
  - 炮：走直线，隔子吃（翻山）
  - 象/相：走"田"字，不过河，注意塞象眼
  - 士/仕：斜走一格，不出九宫
  - 将/帅：直走一格，不出九宫，将帅不能对面
  - 兵/卒：未过河只能前进，过河后可左右平移，不能后退
- [ ] 了解胜负判定：将死、困毙、认输
- [ ] 了解基本禁手：长将、长捉等

### 第二阶段：基本杀法（2-4周）
- [ ] 单车杀法（白脸将、铁门闩）
- [ ] 双车杀法（双车挫）
- [ ] 马后炮杀法
- [ ] 卧槽马杀法
- [ ] 挂角马杀法
- [ ] 重炮杀法
- [ ] 天地炮杀法
- [ ] 闷宫杀法（闷杀）
- [ ] 车马冷着杀法

### 第三阶段：基本残局（1-2个月）
- [ ] 单车胜单士/单象

- [ ] 炮兵胜士象全
- [ ] 车兵对车的基本定式
- [ ] 车马对车的基本定式

### 第四阶段：开局入门（1-2个月）

  - 中炮对屏风马

- [ ] 飞相局
- [ ] 仙人指路

- [ ] 了解先手与后手的区别和应对思路

### 第五阶段：中局提高（持续学习）

## 二、核心提高要点

### 1. 子力价值认知
| 棋子 | 相对价值 | 说明 |
|------|---------|------|
| 车   | 9       | 最强子力，一车顶半盘 |
| 马   | 4       | 开局略弱，残局灵活 |
| 炮   | 4.5     | 开局略强于马，残局弱于马 |
| 士   | 2       | 防守核心 |
| 象   | 2       | 防守辅助 |
| 兵   | 1-2     | 过河卒顶半个车 |

### 2. 十大实战原则
1. **出子要快**：开局迅速出动大子（车马炮），抢占好位
2. **车要通路**：车忌闷在角落，越早出车越好
3. **马要活路**：避免蹩马腿，注意马的出路
4. **炮要有架**：炮需要炮架才有攻击力，注意保持架子
5. **不动老将**：将帅轻易不动，暴露后容易被攻
6. **士象齐全**：防守时尽量保持士象完整
7. **弃子有据**：每一步弃子都要有明确目的
8. **先手意识**：争取主动权，用威胁性着法保持先手
9. **全局观念**：不要只看局部，随时关注全盘
10. **算路要深**：至少看三步棋，重要时刻看五步以上

### 3. 常见错误与改正
| 常见错误 | 改正方法 |
|---------|---------|
| 贪吃丢子 | 吃子前先想对方能否反吃或将军 |
| 只攻不守 | 每走一步先检查自己的将是否安全 |
| 忽视全局 | 走棋前扫一遍全盘，不只盯局部 |
| 开局慢出车 | 优先把车走出来，占据开放线 |
| 残局不会赢 | 系统练习基本残局定式 |
| 马路不清 | 多练习马的行走，注意蹩腿判断 |

## 三、练习方法建议

### 每日练习
- **做杀法题**：每天 5-10 道，训练战术眼光
- **打残局谱**：每天 1-2 个基本残局
- **下实战棋**：至少 1-2 盘，下完复盘

### 复盘习惯
1. 回顾开局：是否出子顺畅？有无明显失误？
2. 分析中局：关键转折点在哪？兑子是否合理？
3. 审视残局：残局处理是否正确？有无更好走法？

### 推荐学习资源
- 《适情雅趣》：经典残局谱
- 《橘中秘》：经典古谱
- 《梅花谱》：经典古谱
- 天天象棋 App：在线对弈与练习
- 象棋巫师：电脑打谱与分析
