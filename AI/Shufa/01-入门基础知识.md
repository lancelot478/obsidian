# 楷书硬笔书法·入门基础知识

## 一、工具准备

| 工具 | 建议 |
|------|------|
| **笔** | 0.7mm 中性笔（推荐百乐 juice/三菱 UM-151）；入门不建议用钢笔，中性笔出墨稳定、容错率高 |
| **纸** | 米字格练字本（田字格亦可），纸张不要太光滑，略有涩感为佳 |
| **字帖** | 推荐田英章《硬笔楷书技法》或荆霄鹏《硬笔楷书入门》；先临摹，后对临 |
| **坐姿辅助** | 桌椅高度合适，光线充足，可在桌面垫一块书写垫板 |

## 二、正确的坐姿与执笔

### 坐姿要领——"头正、身直、臂开、足安"

- **头正**：头部端正，微微前倾，眼睛距纸面约 30cm
- **身直**：上身坐直，胸口离桌沿一拳距离
- **臂开**：两臂自然放在桌面，左手按纸，右手执笔
- **足安**：双脚平放在地面，与肩同宽

### 执笔要领——"三指执笔法"

1. **拇指**：笔杆靠在拇指第一关节内侧，拇指向前方（笔尖方向）用力
2. **食指**：第一关节弯曲，从外侧向内压住笔杆，与拇指相对形成钳状
3. **中指**：第一关节从下方托住笔杆，起辅助作用
4. **无名指与小指**：自然弯曲，抵在纸面上起支撑作用

> **关键点**：
> - 拇指与食指的捏合点距笔尖约 2.5–3cm
> - 笔杆靠在食指第三关节（虎口处），与纸面呈 45°–60° 角
> - 握笔不要太紧，"握鸡蛋"的力度——能控制但不僵硬

## 三、基本笔画（楷书八大基本笔画）

楷书的所有汉字都由以下基本笔画组合而成，练好笔画是写好字的根基。

### 1. 横（一）
- **起笔**：向右下方轻顿
- **行笔**：向右平稳推进，略带上扬（左低右高约 5°）
- **收笔**：到达终点轻顿后回收
- **口诀**：起笔顿、行笔稳、收笔驻

### 2. 竖（丨）
- **悬针竖**：起笔顿，向下直行，末端逐渐提笔出锋（如"中"字）
- **垂露竖**：起笔顿，向下直行，末端顿笔回收（如"十"字）
- **口诀**：竖要直，如立标杆

### 3. 撇（丿）
- **起笔**：向右下方重顿
- **行笔**：向左下方渐行渐轻
- **收笔**：出锋收尾，干净利落
- **口诀**：重入轻出如柳叶

### 4. 捺（㇏）
- **起笔**：轻入
- **行笔**：向右下方逐渐加重
- **收笔**：到最低点后向右平推出锋（"捺脚"）
- **口诀**：轻入重按平出锋

### 5. 点（丶）
- **右点**：向右下方顿笔，圆润饱满后收笔
- **左点**：向左下方轻按后回收
- **口诀**：小如瓜子重如山

### 6. 提（㇀）
- **起笔**：向下顿笔
- **行笔**：向右上方快速提起出锋
- **口诀**：顿后斜上快出锋

### 7. 折（㇕）
- 横折、竖折等，关键在**折角处**要有明显的顿笔转向
- **口诀**：行至折处先顿笔，转向清晰不含糊

### 8. 钩（亅）
- 竖钩、横钩、斜钩等，关键在**出钩前**蓄力顿笔
- **口诀**：驻笔蓄力弹出钩

## 四、笔画练习的核心原则

1. **慢写**：初学阶段追求质量而非速度，一笔一画认真写
2. **观察**：写之前先看字帖 5 秒，观察笔画的起止位置、角度、长短
3. **对比**：每写完一个字，与字帖逐笔对比，找出差距
4. **重复**：同一个笔画/字至少练 20 遍以上，形成肌肉记忆
5. **少而精**：每次练习不贪多，5–8 个字足矣，但每个字要写好

## 五、间架结构基本规律

掌握笔画后，需要学习如何把笔画组合成一个端正的字。

| 规律 | 说明 | 例字 |
|------|------|------|
| **横平竖直** | 横画微微上扬，竖画挺直不歪 | 十、千 |
| **重心平稳** | 字的重心在中心偏上位置 | 大、人 |
| **主笔突出** | 每个字有一个最长/最重要的笔画要写舒展 | 土（长横）、中（长竖） |
| **收放有度** | 笔画有收有放，不能全收或全放 | 林、从 |
| **穿插避让** | 多笔画组合时要相互穿插、避让，不拥挤 | 做、谢 |
| **左收右放** | 左右结构的字，左边偏旁略收，右边部分舒展 | 明、持 |
| **上紧下松** | 上下结构的字，上半部分紧凑，下半部分舒展 | 思、景 |

## 六、初学常见问题

| 问题 | 原因 | 改正方法 |
|------|------|----------|
| 字歪歪扭扭 | 笔画角度不对，缺少基准线意识 | 用米字格练习，注意横画保持一致倾斜度 |
| 笔画粗细不分 | 用力太均匀，没有提按变化 | 刻意练习起笔顿、行笔轻、收笔驻的节奏 |
| 字太大或太小 | 对格子的空间感不够 | 先在米字格中写，让字占格子的 2/3 左右 |
| 握笔太紧手酸 | 执笔方法不对或太紧张 | 放松手指，只用三指控笔，其余手指自然弯曲 |
| 写字太快 | 急于求成 | 放慢速度，每一笔都要有起行收的过程 |
