# 第一卷：空间站——Hello World（初始化）

---

## 第一章 凌晨三点的 Bug

陆辰觉得自己这辈子最大的 bug，就是当了程序员。

凌晨三点零七分，深圳南山区某互联网大厂的工位上，陆辰顶着两个黑眼圈盯着屏幕上密密麻麻的代码，感觉自己的人生正在经历一次严重的内存泄漏——精力在不断流失，却找不到是哪里出了问题。

"陆哥，你还没走啊？"隔壁工位的实习生小王探出脑袋，脸上那种"我先润了你随意"的表情毫不掩饰。

"你猜。"陆辰头也不抬，手指在键盘上噼里啪啦地敲着，"你先走吧，我这还有个 P0 级的线上事故没处理完。产品经理说了，今晚不修好明天就把我的工牌回收了。"

"那不就是路易十六上断头台——脖子再硬也没用嘛。"小王一边收拾背包一边感慨，"辰哥，你说你加班三天了，图啥呢？"

"图个猝死赔偿金比较多。"陆辰面无表情地说。

小王被这话噎了一下，悻悻地走了。

工位上重新安静下来，只剩下机械键盘的敲击声和空调出风口的嗡嗡声。陆辰揉了揉酸涩的眼睛，目光无意间扫到桌面右下角最小化的游戏客户端——崩坏：星穹铁道。

这是他为数不多的精神寄托。每次加班到崩溃的时候，他就会在心里默默想象自己是星穹列车上的旅客，在银河间穿梭，而不是在这破格子间里给资本家当牛马。

他上周刚打完匹诺康尼的主线，正准备等新版本更新。

"唉——"陆辰长叹一口气，伸了个懒腰，脊椎发出一连串令人揪心的咔咔声。他盯着屏幕上的报错信息，忽然觉得一阵恍惚。

代码在他眼前变得模糊，那些字符像是活了过来，扭曲、重组、排列成他从未见过的结构。

"什么情况……是我眼花了还是代码在自己 refactor？"

他使劲眨了眨眼，发现屏幕上的画面已经完全变了——不是 IDE 的深色主题，而是一片浩瀚的星空，无数光点在深邃的虚空中闪烁，像极了……

"等等，这不是星穹铁道的加载画面吗？"

他下意识地嘟囔了一句："要是能进去把这破代码重构一下就好了。"

话音未落，屏幕猛地亮了一下——不是正常的亮，是那种能把整个办公室照得雪白的、几乎要把人眼球灼烧掉的亮。

陆辰感觉自己像被一只无形的大手攥住了后领，整个人被猛地拽进了屏幕里。

失重感。

旋转。

代码碎片像暴风雪一样呼啸着从他身边掠过。

然后，他什么都不知道了。

---

再次睁开眼的时候，陆辰的第一个念头是：工位的地面什么时候变得这么凉了？

他趴在一条金属走廊上，脸贴着冰凉的地板。空气中有一股奇怪的味道——不是办公室里咖啡和外卖混合的气息，而是某种清冽的、带着臭氧味的味道，像是大型机房的气味。

"我是不是加班猝死了？"他爬起来，茫然地打量四周。

走廊两侧是流线型的金属墙壁，每隔几米就有一扇椭圆形的舷窗。透过舷窗，他看到了——

星星。

不是城市夜空中那几颗可怜巴巴的光点，而是铺天盖地的、密密麻麻的、璀璨到令人窒息的星海。一颗巨大的蓝色星球悬在视野的右下方，缓缓旋转着，表面覆盖着厚厚的云层。

陆辰整个人僵住了。

他的大脑飞速运转了大约三秒钟，然后得出了一个结论：

"卧槽，这特效比 4090 渲染的还真。"

紧接着他又得出了第二个结论：

"不对，这不是特效。"

他伸手摸了摸舷窗——冰凉的、真实的、带着微微震动的金属触感。他又掐了一下自己的胳膊——疼。他摸出手机——没信号，但手机是真的，屏幕还停在 IDE 的页面上。

"好家伙。"陆辰愣愣地站在舷窗前，嘴唇动了动，"我这是……穿越了？"

他环顾四周，那些流线型的金属装饰、走廊尽头若隐若现的全息指引标识、空气中弥漫的那种说不出的科技感——他对这一切太熟悉了。

"黑塔空间站？"

话还没说完，远处忽然传来一声巨响。整条走廊剧烈震动了一下，头顶的灯光闪烁了几下，一个刺耳的警报声骤然响起：

**「警告：反物质军团入侵。所有非战斗人员请立即前往安全区域。重复——」**

陆辰："……"

他慢慢地，一字一顿地说："你们这个世界……能申请退货吗？"

没人回答他。

又一声爆炸，这次更近了。走廊尽头的舱门被一股黑紫色的力量轰开，几个银白色的人形机械体破门而入——反物质军团的前锋部队。

陆辰的瞳孔猛地收缩。

他在游戏里见过这些东西无数次，但当它们真实地出现在面前时，那种扑面而来的压迫感和杀意完全不是屏幕里能传达的。每一个机械体都有两米多高，表面包裹着暗紫色的反物质护甲，眼眶中射出冰冷的光芒。

夏侯惇吃眼珠子——这场面，含都含不住了。

"跑！"陆辰的身体比大脑反应更快。他转身就跑，格子衫的下摆在空气中猎猎作响。

他一边跑一边在心里疯狂吐槽：别人穿越是豪门公子、绝世高手，我穿越第一秒就在逃命？这什么开局，新手村就遇 Boss？这需求谁提的，我要看看产品文档！

身后传来金属碰撞的声响，那些机械体正在加速追击。

陆辰在走廊里跌跌撞撞地跑着，连拐了三个弯。他的体能——作为一个每天坐十五个小时的程序员——可想而知，没跑出一百米就开始喘得像个破风箱。

"我……我需要……一个负载均衡器……"他扶着墙大喘气，"把我的心跳频率……分流一下……"

就在他觉得自己快要被追上的时候，前方的走廊拐角处忽然亮起一道耀眼的冰蓝色光芒——

"哈！看这边！"

一个清脆的女声响起，紧接着一支巨大的冰箭从拐角呼啸而出，精准地命中了追在最前面的那个机械体。冰霜瞬间蔓延，将机械体冻成了一座冰雕。

陆辰抬头一看。

一个扎着双马尾的少女站在走廊拐角，手持一把冰蓝色的弓，明媚的大眼睛里满是战意。她穿着一身蓝白相间的服装，整个人像是从冰雪中走出来的精灵。

陆辰的大脑在一瞬间完成了人脸识别。

三月七。

"你……你怎么站在这里发呆！"三月七冲他喊道，一边又拉弓射出一箭，冻住了第二个冲过来的机械体，"快过来！这边安全！"

陆辰的腿比脑子先动了。他连滚带爬地冲过拐角，差点一头栽进三月七怀里——当然被对方灵巧地闪开了。

"你是空间站的研究员吗？"三月七一边掩护一边问，"你怎么一个人在这个区域？"

"我……"陆辰喘着气，大脑飞速运转。他总不能说"我是从另一个宇宙穿越来的程序员，而且我还玩过你的游戏"吧？

"我是……新来的实习生。"他说出了一个无比符合自身气质的谎言。

## 第二章 这个项目组有点东西

三月七显然没什么时间追问一个"实习生"的来历——因为更多的反物质军团单位正从走廊深处涌来。

"先跑再说！丹恒他们在前面的枢纽区等我！"三月七拉着陆辰就往前冲。

陆辰一边被拽着跑，一边偷偷打量身边的少女。三月七比游戏里看起来更加……怎么说呢，更加生动。她的表情变化很快，说话时眉飞色舞，跑动时马尾一甩一甩的，浑身上下都散发着一种感染力极强的活力。

"三月七，对吧？"陆辰脱口而出。

三月七猛地停下来，用看怪物的眼神看着他："你怎么知道我的名字？"

陆辰心里一个咯噔：嘴瓢了。

"呃……你的名牌。"他指了指三月七腰间——那里确实别着一个小小的列车成员标识。

三月七低头看了一眼，将信将疑："你眼神还挺好，跑这么快都能看清。"

陆辰在心里擦了一把冷汗。行，以后说话得过脑子，不然就跟在线上环境直接改数据库一样——一个不小心就全完了。

两人又跑了一段路，终于到达了一个开阔的枢纽区域。

陆辰一进去就看到了另外两个让他瞳孔地震的身影。

一个银发金眸的青年正站在平台中央，手持一柄长枪，动作行云流水地将三个冲过来的反物质单位挑飞。他的表情从始至终都没有变化，冷淡得像是一台精密运转的机器。

丹恒。

而在丹恒旁边——

一个金色短发的少年/少女（陆辰实在分不清开拓者的性别设定，毕竟原版游戏里可以选）正挥舞着一根棒球棍形状的武器，虽然姿势略显生疏，但每一击都带着一种不讲道理的蛮力。

开拓者。

"三月七，你回来了。"丹恒的目光扫过陆辰，淡淡地问，"这人谁？"

"在走廊里捡的！"三月七把陆辰往前一推，"他说他是空间站的实习生。"

丹恒上下打量了陆辰一眼。那个眼神让陆辰想起了 code review 时最严格的那个同事——冷静、精准、一眼就能看出你代码里藏了多少屎。

"穿着很奇怪。"丹恒评价道。

陆辰低头看了看自己的格子衫、牛仔裤和运动鞋，嘴角抽了抽。在这个遍地披风铠甲的世界里，他这身打扮确实显得像个走错片场的群演。

"我……比较注重实用主义穿搭。"陆辰硬着头皮说。

"好了好了，别管穿什么了！"三月七指着远处又一波涌来的敌人，"先解决这些家伙再说！"

丹恒和开拓者立刻进入战斗状态。

陆辰被安排在最安全的角落——他此刻手无寸铁，体能又跟个废物似的，能做的只有蹲在掩体后面当啦啦队。

但他发现自己根本当不了啦啦队，因为他看不懂。

不，他"看不懂"不是说不理解战斗——他在游戏里打过无数次了。而是他发现自己根本无法像在游戏里那样冷静地分析局势。

在游戏里，反物质军团的小怪就是几个数据：HP、攻击力、弱点属性。一清二楚，无非就是选对角色、按对技能。

但在现实中——

那些黑紫色的能量冲击波从他头顶呼啸而过时，气浪差点掀翻他。

丹恒的长枪刺穿机械体时发出的金属碎裂声震得他耳膜发疼。

三月七的冰箭命中目标时迸溅的碎冰打在他脸上，冷得他直哆嗦。

"这什么体验啊！"陆辰抱着头蹲在掩体后面，"之前在游戏里觉得这些小怪随便打打的，现在才知道什么叫站着说话不腰疼。夏侯惇看战场——我可算是独具慧'眼'了！"

一个机械体被丹恒击飞后砸在陆辰旁边的墙上，整面墙被砸出一个人形凹坑。陆辰跳起来，差点原地升天。

"玩了三年星穹铁道，今天终于知道什么叫沉浸式体验了。"他碎碎念着，"我谢谢米哈游的真实还原了啊！"

战斗持续了大约十分钟。丹恒和开拓者的配合默契到令人发指，一个远攻牵制，一个近身收割，三月七则在后方提供冰系控制和掩护。很快，这一波反物质单位就被清理干净了。

"还好，都是些低级单位。"丹恒收回长枪，甚至没有出汗。

三月七转头看陆辰，发现这位"实习生"正紧紧贴着墙壁，脸色发白但眼睛却亮得出奇。

"你没事吧？"三月七担心地问。

"没事没事。"陆辰从墙上把自己撕下来，拍了拍格子衫上的灰，努力让自己看起来像个镇定的成年人，"就是……有点晕。你们这个 PVE 难度比我预想的高一点。"

"PVE 是什么？"三月七歪头。

"呃，就是……打怪。"

丹恒投来一个"你在说什么鬼话"的眼神。

开拓者倒是笑了笑，伸出手想拉陆辰起来。陆辰握住那只手的瞬间，心中涌起一种说不清道不明的感觉——这真的是游戏里那个沉默的主角吗？握手的力道温暖而真实，完全不像一个"可以选择性别的虚拟角色"。

"你好。"开拓者的声音平和而真诚，"我叫星。你叫什么？"

"陆辰。"他回答，"带'大陆'的'陆'，'星辰'的'辰'。"

"陆辰。"星念了一遍，点了点头，"很好听的名字。"

三月七在旁边起哄："哦——你们这就互相介绍上了？我刚才可是救了他命的人，他还没跟我正式打招呼呢！"

陆辰连忙拱手："谢谢三月七姐姐救命之恩。"

"谁是你姐姐！我看起来有那么老吗？！"

"那……三月七同学？"

"更奇怪了！"

丹恒面无表情地打断了这段混乱的对话："我们没时间闲聊。空间站的情况比预想的严重，反物质军团的主力还没有出现。我们需要尽快和姬子会合。"

"同意。"陆辰突然正经起来，"如果我没猜错的话，反物质军团这次的目标是空间站的核心区域——他们不是来搞破坏的，是来拿东西的。"

三人同时看向他。

丹恒的眼神变了——从"随便一个路人"变成了"需要警惕的对象"。

"你怎么知道？"丹恒的语气冷了几度。

陆辰在心里骂了自己一句。又嘴瓢了！这就是直接读了别人代码的结果——你知道答案，但你没法解释你是怎么知道的。

"我……我在空间站的内部通讯里看到过一些文件。"他硬着头皮编，"我虽然是实习生，但是负责数据分析的，看过一些敏感资料。反物质军团最近的活动轨迹指向了空间站存放某些特殊物品的区域。"

这个解释勉强说得通——至少三月七买账了。丹恒还是那个"code review 最严格的同事"的眼神，但至少没有追问。

"走吧。"丹恒转身带路，"不管你说的是不是真的，我们确实不能让反物质军团接近核心区域。"

陆辰跟在三人身后，心跳还没完全平复。

他一边走一边在脑子里飞速整理信息：他现在处于星穹铁道主线第一章——黑塔空间站事件。反物质军团入侵，目标是空间站里存放的某个星核。开拓者（星）会在这里觉醒力量，最终击退入侵者，然后加入星穹列车。

但有一个问题——他自己。

他不在原剧本里。

一个穿着格子衫的程序员出现在黑塔空间站的走廊上，这件事本身就是一个巨大的 bug。他的存在会不会导致蝴蝶效应？会不会把原本的剧情搅得面目全非？

更关键的是——他现在手无寸铁，战斗力约等于零。在游戏里，他可以躺在椅子上指点江山；在现实里，他连一个最低级的反物质单位都打不过。

他现在能做的，只有脖子一缩，祈祷刀别落在自己头上。

"先活下去再说。"陆辰深吸一口气，小声嘟囔，"把这个版本先稳住，优化的事以后再说。"

他不知道的是，此时此刻，在他的双眼深处，有什么东西正在悄然苏醒——那是一行极其微弱的光芒，像是一段被注释掉的代码，正在等待被取消注释的那一刻。

## 第三章 新手保护期不存在的

和开拓者一行人汇合后，陆辰的处境并没有改善太多——准确地说，反而更尴尬了。

因为他发现自己在这个团队里的定位是：完全没有定位。

星在前面开路，拿着棒球棍一路莽过去，每一击都带着某种说不清的力量。三月七在中间，冰箭和护盾切换自如。丹恒断后，长枪翻飞，优雅得像在跳一支致命的华尔兹。

而陆辰呢？

他在最中间，被三个人保护着，像个人形行李箱。

"你能不能别踩我的脚？"三月七第三次被陆辰绊到后终于忍不住了。

"对不起对不起……我这个人跑步协调性不太好。"陆辰连连道歉，"我平时的运动量就是从工位走到茶水间。"

"茶水间是什么？"三月七好奇。

"就是……一个提供咖啡因补给的后勤站点。"

丹恒头也不回地说了句："你到底是做什么的？"

"写代码的。"陆辰老老实实地回答——这是他说过的最真实的话。

"代码是什么？"三月七又问。

陆辰沉默了一秒钟。他发现在这个宇宙里解释"编程"是一件极其困难的事情。

"就是……用一种特殊的语言来控制机器的行为。告诉机器应该做什么、怎么做。"他想了想，补充道，"你可以理解成——我是一个给机器写命令的人。"

"哦！"三月七恍然大悟，"那不就是指挥官吗？只不过你指挥的是机器？"

"差不多吧，但我指挥的机器经常不听话。"陆辰苦笑，"而且出了问题还得我自己背锅。"

"那你现在能指挥什么机器吗？"丹恒忽然问了一个极其现实的问题。

陆辰看了看自己空空如也的双手。

"……不能。"

丹恒没再说话，但那个眼神的意思很明确：废物。

陆辰很想反驳，但他确实反驳不了。在这个用拳头和超能力说话的世界里，一个只会写代码的程序员确实就是废物。

正当他沉浸在自我怀疑中时，一行人来到了一个岔路口。两条走廊分别通往不同的方向，全息指引系统已经因为空间站被入侵而失效了。

"左边通往主控室，右边通往后勤区。"丹恒判断了一下方向，"姬子应该在主控室附近。"

"走左边。"陆辰忽然说。

三人看向他。

"右边那条走廊的通风口有异常。"陆辰指了指右侧走廊天花板上的通风格栅，"你们看，那个格栅的边缘有一层很淡的紫色痕迹——是反物质侵蚀的残留。说明右边那条路已经被反物质军团渗透过了，可能有伏兵。"

三月七凑过去仔细看了看："好像……确实有一点紫色？但也太淡了吧，你是怎么看到的？"

陆辰自己也说不清楚。他只是觉得自己的眼睛在刚才的逃命过程中似乎变得格外敏锐了。那层紫色的痕迹在别人看来可能微不可察，但在他眼中却清晰得像是有人用荧光笔画了个圈。

丹恒盯着那个通风口看了几秒，然后扔了一块碎片过去。

碎片刚飞过通风口下方，嗖的一声，一根黑紫色的触须从格栅缝隙中猛地射出，将碎片击碎！

三月七倒吸一口凉气。

丹恒转头看陆辰的眼神终于有了一点变化——从"废物"升级成了"有点用的废物"。

"走左边。"丹恒说。

陆辰得意了那么一小会儿。这大概是他穿越后第一次觉得自己不完全是个累赘。

虽然他心里也在嘀咕：刚才那个，到底是自己观察力强，还是真的有什么奇怪的事情在发生？

他的眼睛……好像不太一样了。

---

一行人沿左侧走廊前进，很快遇到了更多散落的反物质军团单位。小规模的遭遇战不断发生，但在三个战力天花板面前，这些低级单位基本属于"送经验的小怪"。

陆辰在几次战斗中又发现了一些奇怪的事情。

比如，当三月七的冰箭命中目标时，他似乎能看到一条极其微弱的蓝色光线从冰箭中蔓延出去，形成某种复杂的纹路。

再比如，丹恒挥枪时，枪尖划出的轨迹似乎会在空中留下一层转瞬即逝的绿色光晕。

而开拓者星每次挥击那根棒球棍，虽然动作看起来最"朴素"，但陆辰隐约感到有一种说不出的力量在每一击中涌动——那种力量是金色的、温暖的，像是清晨的第一缕阳光。

"我是不是眼花了……"陆辰揉了揉眼睛，"还是说加班加太多，出现幻觉了？"

他试着集中注意力看向一面普通的金属墙壁。

什么都没有。

他又看向一个被丹恒击倒的机械体残骸。

还是什么都没有——等等。

他集中精神，使劲地盯着那堆残骸看——

一瞬间，一行模糊的文字闪过他的视野：

```
[实体: 反物质军团·前锋单位]
[状态: 已损毁]
[原始HP: 1200]
[攻击力: 85]
[弱点属性: 物理, 火]
```

陆辰的瞳孔猛地缩成针尖大小。

"我……"

他使劲闭了一下眼，再睁开。文字消失了。他再次集中注意力——文字又浮现了，虽然模糊得像隔着一层磨砂玻璃，但确实存在。

"不是幻觉。"他的声音有点发抖。

他真的能看到某种类似游戏面板的东西。

但不是游戏UI——更像是某种底层数据的可视化。模糊、不稳定、闪烁不定，就像一个还在 beta 阶段的功能。

陆辰深吸一口气，让自己冷静下来。

作为一个写了十年代码的程序员，他的第一反应不是"天啊我拥有超能力了"，而是——

"这东西跑在什么框架上的？底层用的什么语言？性能怎么样？能不能调参？文档在哪？"

职业病。

"喂，你站在那发什么呆？"三月七的声音把他拉回现实，"前面快到主控室了，别掉队！"

"来了来了……"

陆辰快步跟上，心中翻江倒海。

他穿越了。他来到了星穹铁道的世界。而且他似乎获得了某种……能看到这个世界"源代码"的能力。

虽然这个能力现在还跟个 alpha 版本似的，功能不全还经常闪退。

但它是真实存在的。

"先给这个能力起个名字吧。"陆辰默默想着，"叫什么好呢……"

他看了看自己的双手，又看了看那些在视野边缘若隐若现的代码碎片。

"编译者之眼。"

嗯，就叫这个。

虽然比起"天眼""法眼""轮回眼"之类的中二名称，这个名字土了不止一个档次。

但他是程序员。

程序员的金手指，就该有程序员的命名风格。

## 第四章 姬子的咖啡有点烫

主控室的大门在他们面前缓缓打开时，陆辰闻到了一股咖啡的香味。

在整个空间站都在被反物质军团蹂躏的时刻，有人在喝咖啡。

这份淡定让陆辰肃然起敬——作为一个经历过无数次线上事故的程序员，他太知道这种"天塌了当被子盖"的心态有多珍贵了。

主控室里，一个红发的女性正坐在指挥台前，一手端着咖啡杯，一手在全息操控面板上快速操作。她的气质沉稳而从容，像一位身经百战的项目经理——不，是那种真正扛过事的技术 leader。

姬子。

"姬子前辈，我们来了！"三月七挥了挥手。

姬子抬头，先是对三人点了点头，然后目光落在陆辰身上，微微挑眉。

"新面孔？"

"在走廊里捡的！"三月七开心地说，似乎很享受"捡人"这个说法。

"我不是物件谢谢。"陆辰弱弱地抗议了一声。

姬子放下咖啡杯，站起来走到陆辰面前。她比陆辰想象中要高一些，气场也强很多——那种"一切尽在掌控"的气场。陆辰见过不少领导，但很少有人能像姬子这样让人觉得既可靠又亲切。

"你好，我是姬子，星穹列车的导航员。"她的语气温和但带着审视，"你是空间站的工作人员？"

"算是吧。"陆辰含糊其辞，"我叫陆辰，是个……技术人员。"

"什么方向的技术？"

"信息处理和逻辑分析。"——这大概是"程序员"在这个宇宙里最体面的翻译了。

姬子似乎接受了这个说法，但她的目光里有一丝探究。陆辰注意到她的眼睛是琥珀色的，温暖而深邃，像是一杯上好的威士忌。

"空间站的情况不太乐观。"姬子转身回到指挥台，调出全息地图，"反物质军团的主力正在向核心存储区推进。黑塔的自动防御系统还在运转，但恐怕撑不了太久。"

全息地图上，空间站被标记为红色的区域越来越多——那些是已经被反物质军团占领或正在交战的区域。核心存储区在空间站的最深处，周围的防御工事正在被一层层剥开。

陆辰盯着全息地图，心跳加速了。

这和游戏里的剧情一致——反物质军团的目标确实是核心区。在原版剧情中，开拓者会和大家一起打到核心区，在那里觉醒星核的力量，最终击退入侵。

但他现在看着这张全息地图，心里想的完全不是"接下来的剧情是什么"，而是——

"姬子姐……呃，姬子女士。"他走到全息地图前，犹豫了一下，还是开了口，"你这个防御体系的部署方式有问题。"

整个主控室安静了。

三月七瞪大了眼睛。丹恒面无表情但目光锐利。星好奇地看过来。

姬子端着咖啡杯的手微微一顿，然后平静地说："说来听听。"

她的反应让陆辰稍微安了心——至少不是那种"你一个实习生也配指点江山？"的态度。好领导就是好领导。

陆辰深吸一口气，指着全息地图说："你们现在的防御策略是'层层阻击'——在每一条通往核心区的通道上都部署了防御力量。这没有错，但问题在于，这种策略假设敌人只会从正面进攻。"

他用手指在地图上划了一个圈："看这里——核心区东南方向有一片已经标红的区域，那里三小时前就被占领了，但反物质军团占领之后并没有继续向核心区推进。这不正常。"

姬子的眉头皱了起来。

"如果我是进攻方，"陆辰继续说，"占领了一个战略位置之后不推进，只有一个原因——我在等。等什么？等正面佯攻把你的注意力全部吸引过去之后，从这个位置发起真正的致命一击。路易十六的侍卫看城门——光盯着前面，后脑勺可就危险了。"

他指着那片红色区域和核心区之间的一小段走廊："如果我没猜错，这段走廊的某个位置有一个隐蔽的通道——可能是维修管道、应急通道或者某种被遗忘的旧设计。反物质军团打算从这里打穿。"

主控室里沉默了好几秒。

姬子缓缓放下咖啡杯，走到全息地图前，仔细查看了陆辰指出的区域。然后她调出了一份空间站的结构蓝图——

"他说得对。"姬子的声音严肃了许多，"这里确实有一条废弃的货运通道，在空间站改造时被封闭了。但如果反物质军团拥有某种腐蚀性力量……"

"那层封闭可能已经被侵蚀穿了。"陆辰接上话。

丹恒看陆辰的眼神再次发生了变化——从"有点用的废物"升级成了"虽然是个废物但脑子好使"。

"需要派人去确认。"丹恒说。

姬子点头，然后看向陆辰："你的分析很专业。你确定你只是个实习生？"

陆辰心虚地笑了笑："我说过了，我是搞信息分析的。发现问题是我的本职工作——虽然我平时发现的大多是自己写的 bug。"

三月七一脸茫然："啥是 bug？"

"就是……工作失误。"

"哦！那你经常出工作失误？"

"……能不能不要这么直白。"

姬子嘴角浮起一丝微笑。在这个兵荒马乱的时刻，这个来历不明的年轻人反而给她带来了一点奇怪的安心感。

不是因为他有多强——他显然没有任何战斗力。而是因为他看问题的角度和思考方式非常独特，像一个习惯了从全局视角分析系统漏洞的人。

"好。"姬子做出了决定，"丹恒，你带三月七去陆辰说的那条废弃通道确认情况。星，你跟我去核心区加强正面防御。陆辰——"

"我呢？"

"你留在主控室。"姬子指了指操控台，"你说你擅长信息分析——主控室的监控系统还能部分使用。你盯着全局，有任何异常立刻通知我们。"

陆辰差点当场流泪。

留在后方看监控？这不就是他最擅长的事情吗？盯着 dashboard 看数据、找异常、发告警——他干了十年了！

"收到。"他二话不说就坐到了操控台前，十指在全息键盘上飞速敲击。操作逻辑和他熟悉的系统不完全一样，但底层思路是相通的——数据可视化、异常检测、告警配置。

"这个界面设计得还行，"他嘟囔着，"至少比我们公司那个内部运维平台强。"

姬子在出门前回头看了他一眼，嘴角的笑意更深了一些。

这个年轻人——看起来不太靠谱，一开口就是些没人听得懂的怪话。但坐到操控台前的那一刻，他整个人的气质都变了——专注、敏锐、高效，像一台突然上线的精密仪器。

有意思。

## 第五章 Beta 版金手指

众人分头行动后，主控室里只剩下陆辰一个人。

他坐在操控台前，面前环绕着十几块全息屏幕，每一块都显示着空间站不同区域的监控画面和数据流。空间站的自动防御系统仍在运转——那些是黑塔设计的机械卫兵，虽然战斗力不如真人，但在关键通道上还是能提供一定的阻击。

"好，先搞清楚现在的局势。"陆辰深呼吸一口气，让自己进入工作状态。

他快速浏览了所有监控画面，在脑中构建出空间站的整体态势。反物质军团的进攻路线很清晰——三路并进，正面两路是主力，看起来声势浩大但推进速度刻意放缓。

"果然是佯攻。"他确认了自己之前的判断，"主攻方向就是那条废弃通道。"

他切出通讯频道："丹恒，你们到了吗？"

"快了。"丹恒简洁的声音从通讯器中传来。

"加快速度。我在监控上看到那片区域的反物质浓度在上升——他们可能已经开始行动了。"

"明白。"

陆辰切掉通讯，继续盯着屏幕。

就在这时，他注意到一件事——全息屏幕上的数据流在他眼中似乎变得格外"清晰"了。

不只是清晰。

那些在屏幕上以数字和图表形式呈现的数据，在他的视野中竟然开始浮现出第二层含义。就像他之前看到的反物质军团残骸上的"属性面板"一样，全息屏幕上的数据流旁边也开始浮现模糊的文字——

```
[系统: 黑塔空间站防御网络 v4.7.2]
[状态: 运行中 (损伤率: 34%)]
[活跃节点: 127/200]
[防御效率: 下降中...]
[异常: 检测到 3 个未授权访问点]
```

"又来了。"陆辰盯着那些浮现的文字，心跳加速。

他深呼吸，尝试让自己集中注意力——就像调 debug 模式一样，把"编译者之眼"切换到更深的层级。

一阵眩晕感袭来。他扶住操控台稳住身形，然后看到了更多——

三个"未授权访问点"在空间站的结构图上亮了起来。其中两个在正面战场的区域，显然是反物质军团正面进攻时留下的——没什么威胁。但第三个——

第三个在核心存储区的正下方。

不是陆辰之前发现的那条废弃通道。是另一个更隐蔽的位置。

"卧槽。"陆辰的后背渗出冷汗。

三路佯攻。

不是两路，是三路。

他之前发现的废弃通道只是第二层佯攻——让防御方以为自己识破了敌人的计划，调兵去堵废弃通道，从而忽略真正的致命一击。

夏侯惇丢了眼珠——真正的危险在你以为安全的地方。

陆辰猛地拍下通讯键："姬子！有新情况！核心区正下方有第三条入侵路线！那条废弃通道是第二层佯攻！真正的主攻在地下——他们从空间站的底部结构向上打！"

通讯器里先是一阵杂音，然后姬子的声音传来，带着压制的惊讶："你确定？"

"我在防御系统的数据流中检测到了第三个未授权访问点。"陆辰飞速地说，"位置在核心区正下方 Z-3 层。反物质侵蚀的强度远超其他两个点——这才是主力。"

"丹恒那边——"

"废弃通道也会有敌人，但规模不会太大。真正的大部队在 Z-3。"

通讯器里沉默了两秒。然后姬子果断下令："丹恒、三月七，清理完废弃通道后立刻转向 Z-3 层。星，跟我走！"

陆辰松了一口气——但只松了一秒钟。

因为他在全息屏幕上看到了一个新的问题：从核心区正下方向上打的反物质军团，已经突破了 Z-3 层的外壁。

按照他们的推进速度，大约还有十五分钟就会到达核心存储区。

而姬子和星至少需要二十分钟才能赶到 Z-3。

差五分钟。

"五分钟……"陆辰盯着屏幕上的数字，大脑飞转。

他开始在防御系统中搜索——有没有什么东西可以在 Z-3 层制造五分钟的延迟？

自动机械卫兵？Z-3 层的卫兵在第一波入侵时已经被摧毁了。

关闭通道闸门？Z-3 层的闸门控制系统已经被反物质侵蚀了，从主控室无法操控。

环境控制系统？他看了看 Z-3 层的环境参数——重力、气压、温度都在正常范围内。但如果他能通过主控室修改这些参数……

"我就是说嘛，"陆辰的手指在键盘上飞舞，"环境控制系统的权限管理这么松，是个人都能改参数，这不就是典型的安全漏洞吗？不过现在这个漏洞可以为我所用了——"

他对着操控台噼里啪啦地操作了一通。他的"编译者之眼"在这个时候帮了大忙——那些浮现在数据流旁边的注释让他能更快地理解这个陌生系统的操作逻辑。

三十秒后，Z-3 层的重力系统被他调到了正常值的三倍。

"来啊！反物质也得受力学约束吧？你们在三倍重力下跑跑看？"

全息屏幕上，Z-3 层的反物质军团推进速度骤然下降。那些机械体原本如履平地的步伐变得沉重而缓慢，有几个平衡性差的单位甚至直接被压趴在了地上。

五分钟的缓冲，到手了。

但代价是——

"警告：Z-3 层重力系统过载运行，预计 7 分钟后该区域结构性崩溃。"

陆辰："……行，经典操作。修了一个 bug，引入了另一个 bug。"

他在心里飞速计算：七分钟结构崩溃，姬子和星二十分钟后到达。即便反物质军团被三倍重力拖慢了五分钟，时间线还是对不上。

除非——他能找到方法在姬子到达之前关闭重力过载，同时又不让反物质军团恢复推进速度。

"需要一个更精细的方案。不能用蛮力，得用巧劲。"

他的手指停在键盘上，眼睛半闭，让"编译者之眼"全力运转——

一阵剧烈的头痛。

但在头痛中，他看到了 Z-3 层环境控制系统的"底层代码"。不是全息屏幕上显示的那些操作界面，而是更深层的、维持整个系统运转的逻辑结构。

那些代码古老、优雅、精密，像一座用数学和物理法则搭建的大厦。陆辰一眼就看出这不是普通工程师的手笔——这是黑塔的杰作。

而在那些代码的深处，他发现了一个"彩蛋"——一段被注释掉的功能模块。

```
// TODO: 局部重力梯度控制 - 实验性功能
// 黑塔注：理论验证完毕，但实际测试中存在 0.03% 的误差
// 暂时禁用。完美主义者不接受 0.03% 的误差。
```

陆辰差点笑出声。

黑塔这个人他太了解了——游戏里的黑塔就是这样一个极致的完美主义者。0.03% 的误差在绝大多数场景下完全可以忽略不计，但黑塔觉得不够完美，于是把整个功能注释掉了。

"黑塔姐，你这个完美主义现在可要救我的命了。"

他深吸一口气，在操控台上输入了一串指令——取消注释，启用局部重力梯度控制。

然后他精准地在 Z-3 层的走廊中设置了一个"重力梯度陷阱"：走廊入口处维持三倍重力，出口处恢复正常。反物质军团越往前走，阻力越大；一旦后退，重力又变轻了——但后退就意味着放弃推进。

而最妙的是，这个梯度控制比全区域三倍重力温和得多，不会导致区域结构崩溃。

"重力系统过载警告已解除。当前模式：局部梯度控制（实验性）。"

陆辰靠在椅背上，长出一口气。

全息屏幕上，Z-3 层的反物质军团被死死卡在了走廊中段——往前走不动，往后退不甘心。

"完美。"陆辰满意地看着这一切，"虽然过程有点曲折，写了个 bug 又修了个 bug，但最后结果是对的。这不就是程序员日常吗？"

他看了看时间。

姬子和星大约还有十三分钟到达。

够了。

## 第六章 天才俱乐部的入场券

战斗结束得比陆辰预想的还要顺利。

姬子和星赶到 Z-3 层后，反物质军团的主力正被重力梯度陷阱困在走廊中段动弹不得。星挥舞着棒球棍冲进去，就像一个棒球选手冲进了投球练习场——每个被困住的敌人都是一个完美的慢速球，挨个敲过去就行了。

丹恒和三月七那边的废弃通道也如陆辰所料，只有少量佯攻部队。清理完毕后他们赶来支援，整个核心区的危机在半小时内就被化解了。

"漂亮。"姬子站在核心存储区的门前，确认里面的东西完好无损后，微微松了一口气。她通过通讯器对陆辰说："你的判断都是正确的。那个重力梯度陷阱也很巧妙——是你设计的？"

"是我从黑塔的系统里翻出来一个被注释掉的实验性功能，稍微调了一下参数。"陆辰老老实实地说，"严格来说，核心创意是黑塔的，我只是个取消注释的。"

通讯器里传来姬子轻笑的声音："你很谦虚。能在那么短的时间内理解并应用一个从未接触过的系统，这本身就是一种能力。"

陆辰挠了挠头。说实话，如果没有"编译者之眼"，他不可能做到这一点。那种能看到"底层代码"的能力让他在理解陌生系统时拥有了一个巨大的优势——别人需要读文档、做测试才能理解的东西，他一眼就能看到本质。

但这种能力还非常不稳定。每次使用后他都会感到一阵头痛和眩晕，像是大脑的内存被狠狠地挤占了一样。

"这玩意还在 beta 阶段，"他默默想着，"稳定性堪忧。"

---

危机解除后的空间站进入了紧急修复模式。

陆辰被安排在一个临时休息区里——本来说是让他休息的，但他坐不住，满脑子都在想"编译者之眼"到底是怎么回事。

他正对着一面墙壁发呆的时候，休息区的门打开了。

一个小小的人偶走了进来。

大约三十厘米高，穿着华丽的红色裙装，头上戴着精致的饰品。如果不是它在自己走路，陆辰会以为这是个摆件。

黑塔——的遥控人偶。

"你就是那个在主控室操作重力系统的人？"人偶的声音冷淡而倨傲，带着一种"我百忙之中抽空来见你算你运气好"的气场。

陆辰从椅子上站起来。面对一个只有他小腿高的人偶，他不太确定应该蹲下来还是站着。

"呃……是我。你是黑塔？"

"黑塔的远程人偶。本体没空见你。"人偶抬起头看着他——那个角度让陆辰有点想笑但不敢笑，"我对你做的事情很感兴趣。"

"是因为我取消注释了你的实验性功能？"

"不。"人偶绕着陆辰走了一圈，像是在检阅一件有趣的展品，"我感兴趣的是——你是怎么找到那段代码的。那个模块被封装在系统的第七层底层架构中，没有任何用户界面的入口。即便是空间站最资深的工程师，也无法通过正常的操控台访问到那个层级。"

陆辰心里一紧。

"你到底看到了什么？"人偶停在他面前，微微仰头，那双精致的眼睛里闪烁着不属于机械的光芒——那是黑塔本人的意识在操控。

陆辰沉默了几秒。

他在衡量要不要对黑塔说实话。在游戏里，黑塔是一个绝顶聪明但极度傲慢的天才——她是"天才俱乐部"的成员之一，智力排名宇宙前几。这样的人，你跟她玩心眼没有任何意义，她三秒钟就能看穿你。

但同时，黑塔也是一个纯粹的学者——她对一切有趣的新事物都有着近乎疯狂的好奇心。如果他的能力足够"有趣"，黑塔不但不会为难他，反而会把他当成一个珍贵的研究对象来保护。

"我能看到一些……别人看不到的东西。"陆辰斟酌着措辞，"比如系统的底层结构、数据的运行逻辑，甚至一些被隐藏的代码。"

"具体说。"

"好比刚才的重力系统——在正常的操控界面上，它只显示基本的参数调节功能。但在我眼中，我能看到整个系统的架构图，包括所有被封装、被注释、被禁用的模块。"

人偶站在原地不动了好一会儿。

"有意思。"

黑塔说出了这两个字，然后走到休息区的一面墙壁前。她轻轻敲了三下，墙壁的一小块区域亮了起来，显示出一段极其复杂的方程式。

"看看这个。"黑塔说，"告诉我你看到了什么。"

陆辰走过去，盯着那段方程式。

对普通人来说，这就是一堆复杂的数学符号。但当他集中注意力，让"编译者之眼"启动时——

方程式的背后浮现出了一层新的信息。那不再是简单的"属性面板"，而是一段更深层的东西——某种关于这个方程式"本质"的描述。

```
[类型: 命途·存护 - 基础防御方程]
[创建者: 星神·阿德里安（已陨落）]
[功能: 在目标区域构建空间稳定场]
[状态: 完整]
[权限: 星神级 - 只读]
[注释: 此方程为存护命途的核心逻辑之一。
       通过锚定空间结构的基本参数，
       在指定区域内创建一个"不可被轻易改变"的稳定域。
       ——但"不可被轻易改变"本身，也是一种可以被改变的参数。]
```

陆辰的呼吸停了一秒。

"这是……存护星神留下的东西。"他的声音有些干涩，"一个空间稳定场的基础方程。创建者是阿德里安。状态完整，权限是星神级只读——"

他说到这里停住了，因为他注意到人偶的表情变了。

不是生气，不是惊讶。

是——兴奋。

那种一个科学家发现了全新研究方向时才会有的兴奋。

"你真的能看到。"黑塔的声音不再冷淡，取而代之的是一种压抑着激动的平静，"你能看到命途的底层描述。这不是分析、不是推理，你是直接'读取'了它。"

陆辰点了点头。

"你知道这意味着什么吗？"人偶仰头看着他，"这个宇宙中，能直接读取命途底层信息的存在，除了星神本身——我还从未见过第二个。"

"那我岂不是很牛？"陆辰试图用玩笑缓解紧张气氛。

"不，你很危险。"黑塔的语气忽然严肃了，"能看到命途底层的人，对星神来说要么是珍贵的工具，要么是需要消灭的威胁。你现在还太弱——你看到了不该看的东西，却没有保护自己的能力。这就像——"

她顿了顿，似乎在寻找一个合适的比喻。

"像一个不设防的数据库暴露在了公网上。"陆辰替她说完了。

人偶看了他一眼。

"你的比喻虽然粗俗，但很准确。"

陆辰苦笑了。

他知道黑塔说得对。他现在的状态就是一个裸奔的数据库——有价值的数据一大堆，但没有任何防火墙和权限管理。任何一个有能力的"黑客"都能把他扒个精光。

"我有一个建议。"黑塔的人偶转了个圈，像是在踱步思考，"星穹列车即将离开空间站。跟他们走。"

"为什么？"

"两个原因。第一，星穹列车的成员足以保护你。姬子和瓦尔特的实力在宇宙中也算得上一流，在你成长起来之前，他们是你最好的保镖。"

"第二呢？"

"第二——"人偶停下来，那双精致的眼睛里闪过一丝意味深长的光芒，"你跟着列车旅行，会遇到更多的命途痕迹。而我非常想知道，你那双眼睛在面对不同的命途时，能看到什么。"

"所以你是想让我当你的移动探测器？"陆辰一语道破。

"别说得那么难听。"黑塔的人偶从口袋里掏出一个小小的通讯装置，"这个给你。以后你每到一个新的地方，把你看到的命途信息发给我。作为交换，我会为你提供技术支持——如果你的能力出了什么问题，我可以远程协助你分析。"

陆辰接过通讯装置，翻来覆去地看了看。

"这算不算……学术合作？"

"勉强算吧。"黑塔的人偶转身往门口走去，"虽然你的水平离天才俱乐部差了十万八千里，但你的眼睛足够有趣。在天才俱乐部里，'有趣'本身就是一张入场券。"

她走到门口时忽然停了一下，头也不回地说："对了，你在我的系统里发现的那段注释代码——0.03% 的误差我回头会修掉的。"

"不用，"陆辰脱口而出，"0.03% 在工程实践中完全可以接受。你把精力花在更重要的事情上吧。"

人偶转过头来，用一种奇怪的眼神看着他。

"你说的不对。0.03% 的误差在空间站的日常运行中确实可以忽略。但如果有一天，你需要用到那个功能来对抗星神级别的力量——0.03% 就是生与死的距离。"

说完，她走了。

陆辰站在原地，后背一阵发凉。

他想起了一句话：细节决定成败。在编程领域，一个小小的浮点精度问题可以导致火箭爆炸。

也许黑塔是对的。

在星神的世界里，0.03% 的误差真的可能要命。

他低头看了看手中的通讯装置，然后小心翼翼地揣进了格子衫的口袋。

就在他把通讯装置放好的时候，"编译者之眼"不受控制地闪了一下——在他的视野边缘，黑塔空间站数据库的深处，一行极其模糊的文字一闪而过：

```
// [已注释] function trailblaze_origin() {
//   ...
//   return undefined;
// }
// 注释者: [权限不足，无法读取]
// 注释时间: [数据损坏]
```

一个被注释掉的函数。名字叫"trailblaze_origin"——开拓的起源。

注释者和时间都无法读取。

陆辰还来不及仔细看，那行文字就消失了，取而代之的是一阵剧烈的头痛。他扶住墙壁，额头上渗出冷汗。

"我刚才……看到了什么？"

他不确定那是不是幻觉。

但他有一种直觉——那行被注释掉的代码，比黑塔给他看的存护方程重要一万倍。

他只是还不知道为什么。

## 第七章 「星核」觉醒

空间站的危机并没有真正解除——它只是从第一阶段进入了第二阶段。

反物质军团的第一波攻击被击退后，一个短暂的平静期降临了。但所有人都知道，这不是结束，而是暴风雨前的宁静。

陆辰在主控室的监控系统中发现了令人不安的迹象：空间站外围区域的反物质浓度不但没有下降，反而在持续上升。这意味着反物质军团的主力还没有撤退——他们在积蓄力量，准备发动更猛烈的第二波攻击。

"根据浓度上升曲线推算，"陆辰对着通讯器说，"第二波攻击大概在四十分钟后。而且这次他们不会再分兵了——所有力量会集中在一个点上突破。"

姬子的声音从通讯器中传来："目标还是核心存储区？"

"大概率是。"陆辰皱着眉头，"但有个问题——如果他们四十分钟后全力进攻，以空间站目前的防御力量来看……"

他没有说完，但所有人都明白他的意思。

扛不住。

空间站的防御系统在第一波攻击中已经损伤了接近一半。自动机械卫兵折损过半，多个区域的防御工事被摧毁。而反物质军团的主力几乎完好无损——第一波进攻对他们来说不过是火力侦察。

"我们需要一个计划。"姬子说。

众人在一个临时指挥室里碰了头。姬子、瓦尔特、星、三月七、丹恒，再加上陆辰——一个看起来最不搭的组合。

瓦尔特靠在墙边，双手抱胸。这位来自"另一个世界"的前辈一直保持着沉默，但陆辰注意到他的目光偶尔会落在自己身上——那是一种复杂的、审视的目光。

"正面硬扛是不可能的。"丹恒简洁地说，"我们需要一个能一击制敌的方案。"

"核心存储区里的东西——"姬子犹豫了一下，看向瓦尔特。

瓦尔特微微点头。

"核心存储区里保存着一枚星核。"姬子对众人说，"反物质军团来就是为了它。如果我们能在第二波攻击之前把星核转移到安全位置——"

"来不及了。"陆辰摇头，"四十分钟的时间不够进行安全转移。星核的能量波动太大，贸然移动可能触发连锁反应。而且——"

他看向星。

在原版剧情中，这枚星核最终与开拓者融合，赋予了开拓者独特的力量。这是整个游戏最核心的设定之一。

"如果我们不能移走星核，"陆辰慢慢地说，"那就只能想办法利用它。"

"什么意思？"三月七问。

陆辰看着全息地图，脑子里飞速运转。他知道原版剧情中星核觉醒的触发条件——开拓者在极端危机中与星核产生共鸣，从而觉醒力量。但那是一个"偶然事件"，充满了不确定性。

如果他能利用"编译者之眼"提前分析星核的"代码结构"，找到触发觉醒的条件……

"我需要去核心存储区看一样东西。"陆辰说。

所有人的目光都集中在他身上。

"我知道你们觉得我就是个手无缚鸡之力的技术人员。"陆辰坦然地说，"事实上你们说得对，我确实打不过任何反物质军团的小兵。但我有另一种能力——我能看到一些你们看不到的东西。信息。数据。本质。"

他深吸一口气："让我去看看那枚星核。也许我能找到扭转局势的办法。"

沉默。

然后姬子说："好。我陪你去。"

---

核心存储区是一个巨大的球形空间，漂浮在空间站的最中心位置。

当厚重的合金大门在他们面前打开时，陆辰被眼前的景象震撼了。

球形空间的正中央，一枚拳头大小的晶体悬浮在半空中。晶体表面流动着金色和银色交织的光芒，像一颗微型的恒星。周围的空气似乎被它扭曲了，光线在晶体附近发生了明显的弯折。

星核。

在游戏里，这就是一个发光的道具。但在现实中——

陆辰感觉自己的每一个毛孔都在颤栗。那枚星核散发出的能量波动不是用眼睛"看到"的，而是用整个身体"感受到"的。它像一首无声的交响乐，每一个音符都在拨动他灵魂的琴弦。

"编译者之眼"不由自主地激活了。

这一次，他看到了迄今为止最复杂、最壮观的"代码"。

星核的内部是一个多层嵌套的代码结构。最外层是一层保护性的"封装"——简洁而强固。第二层是能量转换的"函数"——复杂但有序。第三层……

第三层他看不清楚。太深了。每次他试图深入，头痛就加剧一分。

但他看清了足够多的信息：

```
[实体: 星核 (Stellaron)]
[状态: 休眠]
[能量等级: ████ (数据被遮蔽)]
[共鸣条件: 需要与兼容载体建立链接]
[兼容载体条件: 
  1. 无既有命途绑定
  2. 具备承载能力的精神强度
  3. 与星核意志产生共振]
[注意: 强制激活可能导致载体崩溃]
```

陆辰盯着"共鸣条件"看了好一会儿。

无既有命途绑定——星是从"某个地方"被送到空间站的，在与星核融合之前没有任何命途。这个条件他满足。

精神强度——开拓者的精神强度是主角光环的一部分，这个毋庸置疑。

与星核意志产生共振——这才是关键。在原版剧情中，这个共振是在极端危机中偶然发生的。但如果能人为引导……

"我有个想法。"陆辰转向姬子，"但是有风险。"

"说。"

"这枚星核需要一个'兼容载体'来觉醒。星——就是那个金发的孩子——我认为他就是那个载体。"

姬子的眉头微微一动。

"别问我怎么知道的，"陆辰抢先说，"就当我是用数据分析出来的。重点是——如果能引导星与星核建立共鸣，星核觉醒后的力量足以击退反物质军团的第二波攻击。"

"'引导'？怎么引导？"

"我需要一点时间来分析星核的共振频率。"陆辰盯着悬浮的晶体，"如果我能找到共振的触发条件，就可以在第二波攻击到来时精确地引发觉醒——而不是靠碰运气。"

"风险呢？"

"如果我的分析出了偏差，强制共振可能会伤害到星。"陆辰老老实实地说，"而且我的'分析工具'本身还在测试阶段，准确率……我不敢打包票。"

姬子沉默了很久。

"你平时工作的准确率怎么样？"她忽然问了一个奇怪的问题。

"经常写 bug。"陆辰说。

"……"

"但是，"他接着说，"真正关键的项目，我从来没掉过链子。平时写 bug 是因为那些需求太蠢了我不想认真做。"

姬子看着他的眼睛——那双因为过度使用"编译者之眼"而微微充血的眼睛里，有一种奇怪的坚定。

这个年轻人，嘴上没个正经，看起来吊儿郎当，但说到最后那句话时，语气里有一种不容置疑的认真。

像一个平时总迟到早退的员工，在公司真正遇到危机时第一个站出来加班的那种人。

不太靠谱，但很靠得住。

"好。"姬子做出了决定，"我给你二十分钟。分析出来你就上；分析不出来，我们用其他方案。"

"二十分钟够了。"陆辰点头，"帮我叫星过来。我需要先'扫描'一下他。"

---

接下来的二十分钟，是陆辰穿越以来最高强度的"加班"。

他让星站在星核旁边，然后用"编译者之眼"同时观察两者——星核的代码结构和星身上的"属性数据"。

让他惊讶的是，星的"代码"和普通人完全不同。

普通人在他眼中是一套相对简单的数据结构——生命体征、属性面板、模糊的命途连线。但星的代码像一个巨大的、未初始化的容器——内部空空荡荡，却拥有惊人的"存储空间"和"计算能力"。

就像一台崭新的高性能服务器，还没有安装任何操作系统。

而星核——就是那个等待被安装的"操作系统"。

"接口是兼容的。"陆辰喃喃自语，"数据类型匹配，容量足够，权限……需要一个触发器。"

他进一步分析星核的"共振频率"——那是一组极其复杂的参数，像一首宇宙级别的密码。他的"编译者之眼"在高负荷运转下开始出现"画面撕裂"，信息断断续续地闪烁。

头痛欲裂。

鼻血流了下来。

"陆辰！"三月七惊呼。

"没事——"他用袖子抹了一把鼻血，继续盯着星核的代码，"还差一点……"

他看到了。

共振的触发条件不是某个精确的频率值，而是一种"状态"——当载体的意志与星核的意志产生同频共振时，链接就会建立。

而"意志"这个东西，无法量化，无法编程，无法强制触发。

它只能靠——

"星。"陆辰擦着鼻血转过头。

金发的开拓者一直安静地站在旁边，用一种好奇而信任的目光看着他。

"我问你一个问题。"陆辰说，"你想要力量吗？"

星歪了歪头："为什么这么问？"

"因为接下来的战斗，凭我们现在的力量打不赢。"陆辰指着星核，"但如果你愿意的话，这枚星核可以给你力量。巨大的力量。"

"代价呢？"星问。

陆辰一愣。他没想到这个看起来单纯的孩子会问这个问题。

"可能会疼。也可能会改变你——从此以后你就不再是一个普通人了。"

星沉默了一会儿，然后笑了。

那个笑容明亮得像星辰。

"我本来就不是普通人。"星说，"我甚至不知道自己是从哪里来的。但有一件事我很确定——我不想看着这里的人被伤害而无能为力。"

陆辰看着那双金色的眼睛，忽然觉得一阵感动——和代码无关的、纯粹的感动。

"好。"他深吸一口气，"那我告诉你怎么做——"

他没有来得及说完。

空间站再次剧烈震动。

"反物质军团第二波攻击提前了！"丹恒的声音从通讯器中炸响，"他们直接从外壁打穿了防御层！正在向核心区突进！"

四十分钟的预估，实际只过了二十五分钟。

陆辰的脸色变了："该死，他们加速了——"

"预估误差？"姬子快速问道。

陆辰咬了咬牙："是我分析得不够精确。反物质浓度上升曲线不是线性的，是指数级的——我用了线性模型来预估，算少了。我的锅。"

话虽如此，现在不是追究责任的时候。

核心存储区的外层防御已经被突破了。大量的反物质军团单位正在涌向最后一道防线。

"没时间了。"陆辰看向星，"你准备好了吗？"

星看着悬浮的星核，然后看向走廊尽头——那里传来爆炸声和金属碰撞的声响，丹恒和三月七正在拼命阻击敌人。

"告诉我怎么做。"星说。

"走过去，伸出手，触碰它。"陆辰的声音出人意料地平静，"然后——不要抗拒它。把你心里最真实的愿望告诉它。你为什么想要力量，你想要保护什么。它会听到的。"

星点了点头，转身走向星核。

每走一步，空气中的能量波动就更强一分。金色的光芒从星核表面倾泻而出，将整个球形空间照得如同白昼。

星伸出手。

指尖触碰到星核的瞬间——

整个黑塔空间站都感受到了那一刻。

一道金色的光柱从核心存储区冲天而起，穿透了空间站的所有楼层，直刺苍穹。光柱中蕴含的能量让每一个生物都感到了一阵战栗——不是恐惧，而是某种更原始的东西。

敬畏。

陆辰被冲击波推倒在地，但他死死地盯着光柱中心的那个身影。

星悬浮在半空中，周身环绕着金色与银色交织的光芒。那枚星核已经消失了——或者说，它已经成为了星的一部分。

"编译者之眼"在这一刻被动激活，陆辰看到了一幕终生难忘的景象：

星的"代码"——那个原本空空荡荡的巨大容器——正在被星核的力量飞速填充。成千上万行代码如瀑布般涌入，构建出一个全新的、前所未有的系统架构。

那是「开拓」命途的初始代码。

陆辰瞪大了眼睛。他第一次如此清晰地看到一条命途的诞生——不，不是诞生，是重启。这些代码古老得超出他的想象，每一行都带着穿越时空的沧桑感。它们不是新写的，而是从某个极其遥远的地方被"调用"过来的。

就像一个沉睡了无数年的程序，终于被重新编译运行。

"这就是……开拓。"陆辰喃喃道。

光芒消散后，星缓缓降落在地面上。

金发少年睁开了眼睛。

那双眼睛里多了什么东西——不只是力量，还有一种坚定和从容，像是一个刚刚完成系统初始化的新生服务器，清醒地审视着自己即将面对的一切。

"我感觉……很好。"星握了握拳，金色的能量从指间流过，"就像……我终于找到了自己该做的事。"

"恭喜你。"陆辰从地上爬起来，拍了拍屁股上的灰，"系统初始化完毕，欢迎进入正式版。"

星对他笑了笑。

然后转身，面朝那些正在突破最后防线的反物质军团——

迈出了第一步。

## 第八章 线上事故处理

星的觉醒让战局发生了根本性的逆转。

陆辰蹲在核心存储区的一个角落里——是的，他又回到了那个"人形行李箱"的定位——看着星手持新觉醒的武器冲入反物质军团的队伍中，心里五味杂陈。

嫉妒是不可能嫉妒的。主角就是主角，这种"关键时刻天降神力"的待遇他一个写 bug 的 NPC 不配拥有。

但说不感慨那是假的。

看着星挥出第一击时那道金色的弧光——力量、速度、精准度都是觉醒前的几十倍——陆辰真切地感受到了"命途"这两个字的重量。那不是简单的属性加成，而是一种从根本上改变存在层级的蜕变。

就像一个 Hello World 程序忽然被重构成了一个完整的操作系统。

"星那边没问题了。"陆辰通过通讯器说，"但正面战场不是唯一的问题——我在监控里看到一些反物质单位绕过了正面，正在向空间站的能源核心移动。"

"他们想炸掉空间站？"姬子的声音带着一丝凝重。

"不，更像是想切断空间站的能源供应。一旦能源核心停摆，空间站的所有防御系统和生命维持系统都会瘫痪。到时候就算星再能打，也没用——你不能在真空中打架。"

"你能从主控室截住他们吗？"

陆辰飞速跑回主控室——好在这段路他已经跑过好几次了，闭着眼睛都能走。

坐到操控台前，他调出了能源核心区域的监控画面。

七个反物质军团的精锐单位正在沿维修通道向能源核心推进。和之前的杂兵不同，这些精锐个头更大、装甲更厚、移动更快。

"直觉告诉我，这七个玩意不太好对付。"陆辰启动"编译者之眼"扫描了一下——

```
[实体: 反物质军团·歼星单位]
[HP: 8500]
[攻击力: 340]
[弱点属性: 冰, 雷]
[特殊能力: 反物质护盾 (降低 50% 物理伤害)]
[威胁等级: 中高]
```

"HP 八千五？攻击力三百四？"陆辰倒吸一口凉气，"之前那些杂兵才一千二的 HP……这七个怪，丹恒一个人都够呛。"

他迅速扫了一眼维修通道的环境系统。

和之前 Z-3 层不同，维修通道的环境控制非常有限——没有重力调节功能，温控范围也很窄。他没法再用同样的招数。

"换个思路。"

陆辰的目光在监控画面和全息蓝图之间来回扫视。

维修通道的结构很简单——一条长长的直线通道，两侧是管线和维修舱门。通道中段有一个 T 字形岔口，一边通往能源核心，另一边通往……

废液排放系统。

陆辰的眼睛亮了。

"哈！"

他飞速查看了废液排放系统的参数——这个系统负责将空间站日常运转产生的冷却液和废液排放到外太空。排放管道的口径足够大，排放压力也够高。

如果他在那七个精锐单位经过 T 字岔口的时候，突然打开废液排放阀门……

"高压冷却液冲洗。"陆辰的手指在键盘上跳舞，"虽然杀不死它们，但足以让它们滑一跤。冷却液温度零下七十度，冰属性弱点——理论上可以造成额外伤害并降低移动速度。"

他一边操作一边嘟囔："修电脑的时候清理灰尘、通一下管道顺便就把 bug 修了，这不就是我日常工作的写照吗？"

但还有一个问题——即便冷却液冲洗能拖延一段时间，也不能彻底解决这七个精锐单位。他需要有人去收割。

"三月七！"他拍下通讯键，"你在哪？"

"我在——哎呀——"通讯器里传来冰箭射出的声响和三月七的惊叫声，"我在 C 区走廊清理残余敌人！怎么了？"

"能源核心方向有七个精锐单位，我需要你去拦截。别担心，我会先用'特殊手段'削弱它们。"

"什么特殊手段？"

"到了你就知道了。相信我。"

"你上次说'相信我'的时候差点把 Z-3 层的结构搞塌——"

"那是意外！已经修了！快来！"

三月七叹了口气，但还是立刻改变方向，朝维修通道赶去。

陆辰盯着监控画面，计算着时机。

七个歼星单位正在以稳定的速度推进。按照它们的移动速度，大约三分钟后会经过 T 字岔口。三月七全速赶来大约需要四分钟。

差一分钟。

"又是差一分钟的剧本。"陆辰磨了磨牙，"行，那我就再即兴发挥一下。"

他翻了翻维修通道的其他可控系统——照明、通风、舱门……

舱门。

维修通道每隔五十米就有一道应急舱门。这些舱门平时是敞开的，只有在紧急情况下才会关闭。

如果他在精锐单位经过的时候关闭前后的舱门，再打开冷却液冲洗——

"先关门，再冲水。夏侯惇吃饺子——脸上只有一只眼，但饺子一口也不会少——不对，这什么比喻。"陆辰摇了摇头，"总之就是先困住再收拾。"

三分钟后。

七个歼星单位进入了 T 字岔口附近的区域。

陆辰深吸一口气。

"三——"

他的手指悬在控制面板上。

"二——"

监控画面上，七个黑紫色的大家伙正整齐地排成纵队通过走廊。

"一——"

他同时按下了三个控制键。

前后两道应急舱门轰然关闭，将七个歼星单位困在了大约五十米的走廊段中。

与此同时，T 字岔口的废液排放阀门大开——

零下七十度的高压冷却液像一条暴怒的冰河，从侧方管道中呼啸而出，瞬间灌满了整段走廊。

监控画面上，那些原本威风凛凛的歼星单位被冰蓝色的液体冲得东倒西歪。冷却液接触到它们表面的反物质护甲时发出刺耳的嘶嘶声——温度差异造成的热应力让护甲表面出现了密密麻麻的裂纹。

"冰属性弱点生效。"陆辰盯着"编译者之眼"反馈的信息，"护甲完整度下降到百分之四十……百分之三十……好，差不多了。"

他关闭冷却液阀门，然后打开了前方的舱门——正对三月七赶来的方向。

"三月七！它们在里面，护甲已经被冰冻裂了！往死里打！"

三月七到达走廊入口时，看到了七个浑身挂着冰碴子、狼狈不堪的歼星单位正试图从冷却液中爬起来。

她愣了一秒。

"你之前说的'特殊手段'就是……给它们洗冷水澡？"

"高效能环保作战方案。"陆辰一本正经，"快打，冰冻效果持续不了太久。"

三月七虽然满头问号，但战斗直觉让她毫不犹豫地冲了上去。冰弓满拉——在已经被冰冻削弱的敌人面前，她的冰属性攻击伤害直接翻倍。

一箭一个，干净利落。

当最后一个歼星单位倒下时，三月七整个人都没怎么出汗。

"这……也太轻松了吧？"她难以置信地看着遍地残骸。

"兵者，诡道也。"陆辰靠在操控台椅背上，长出一口气，"正面打不过就别正面打。战场上最强的武器从来不是拳头，是信息差。"

"你说的怪有道理的。"三月七通过通讯器说，"虽然我总觉得哪里怪怪的……"

"可能是因为你觉得用冷却液冲人不太体面？"

"对！就是这个！有点像在食堂打架泼汤……"

陆辰笑了。这个比喻虽然离谱但意外地准确。

---

正面战场上，星的觉醒彻底扭转了局势。

金色的光芒在走廊中横扫，每一击都带着「开拓」命途的力量。那些在陆辰眼中拥有恐怖数值的反物质军团精锐，在觉醒后的星面前不堪一击。

姬子和瓦尔特作为后援，负责清理漏网之鱼和保护侧翼。丹恒则像一把锋利的匕首，专挑敌人阵型的薄弱环节下手。

四十分钟后，反物质军团的第二波攻击被彻底击溃。

残余的敌军力量开始向空间站外撤退——但它们撤退的方式很奇怪。不是溃逃，而是有序地后撤，像是接到了某个统一指令。

"它们不是被打跑的。"陆辰盯着监控画面皱眉，"是主动撤退的。就像……程序到达了设定的终止条件。"

"什么意思？"姬子通过通讯器问。

"反物质军团这次进攻的目的——可能从一开始就不只是抢夺星核。"陆辰慢慢地说，"它们的另一个目标，也许已经达成了。"

"什么目标？"

陆辰没有回答。

因为他不知道答案。

在原版游戏剧情中，反物质军团在空间站的这次入侵就是以失败告终的。但现在他身处其中，从"旁观者"变成了"当事人"之后，很多在游戏里被一笔带过的细节变得格外扎眼。

比如——反物质军团为什么要在这个时间点进攻黑塔空间站？它们是怎么知道星核存放在这里的？毁灭星神南纳克的军团虽然强大，但也不会无缘无故地浪费兵力。

这些问题在游戏里不重要——玩家只需要打怪就行了。但在现实中，每一个"为什么"的背后都可能藏着一条通往深渊的线索。

"先不想了。"陆辰揉了揉太阳穴，"先把当前版本稳定下来，遗留问题下个迭代再处理。"

## 第九章 你们这个项目组还缺人吗

战斗结束后的黑塔空间站一片狼藉。

到处是破损的墙壁、碎裂的舷窗和熄灭的全息面板。空气中弥漫着烧焦的气味和冷却液的刺鼻味道。自动修复系统已经开始工作，小型维修机器人在走廊中穿梭忙碌，像一群勤勤恳恳的蚂蚁。

陆辰坐在一个临时休息区的角落里，靠着墙壁，闭着眼睛。

他累坏了。不是体力上的累——他全程都在坐着——而是精神上的累。"编译者之眼"的高强度使用让他的大脑像被拧干的毛巾，每一根神经都在抗议。

"给你。"

一只手递过来一杯热饮。

陆辰睁开眼，看到姬子站在他面前，手里端着两杯咖啡。

"谢谢。"他接过杯子，热度透过杯壁传到掌心，让他舒服地叹了一口气。

"你做得很好。"姬子在他旁边坐下，优雅地啜了一口咖啡，"Z-3 层的重力梯度陷阱、废弃通道的预警、能源核心的冷却液防御——如果没有你，今天的伤亡人数至少翻三倍。"

"我只是坐在后面动动手指而已。"陆辰摇头，"真正在前面拼命的是你们。"

"但战争不只是拼命。"姬子看着远处正在修复的走廊，"信息、判断、时机——这些和战斗力一样重要。有些时候，一个正确的决策比一千拳重击更有价值。"

陆辰没有说话。

姬子看了他一眼，然后笑了："你在想什么？"

"我在想……"陆辰犹豫了一下，"我不属于这里。"

"嗯？"

"我是说……我不是这个世界的人。"

他说出这句话的时候，心里做好了被当成疯子的准备。但姬子只是安静地看着他，没有打断。

"我来自一个很远的地方。一个……没有星神、没有命途、也没有反物质军团的地方。我到这里来是一个意外。我不知道怎么回去，甚至不知道能不能回去。"

他低头看着杯中的咖啡，液面映出他疲惫的脸。

"所以我一直在想——我到底应该怎么定义自己在这个世界的位置。我没有战斗力，不属于任何组织，连这个宇宙的常识都不了解。我能做的就是坐在屏幕后面看看数据、动动脑子。这种程度的贡献……值得你们带上我吗？"

姬子安静地听他说完。

然后她放下咖啡杯，转过身来正对着他。

"陆辰，我问你一个问题。"

"什么？"

"今天的战斗中，你为什么要那么拼命？你可以只做最基本的事情——盯着监控、报告情况。但你做了远不止这些。你分析了敌人的进攻策略，设计了 Z-3 层的陷阱，发现了第三条入侵路线，引导星完成了觉醒，还即兴想出了冷却液冲洗的战术。"

她的目光直视他的眼睛。

"你做这些的时候，有没有想过——你不属于这里，你没必要这么拼？"

陆辰愣住了。

他认真回想了一下。

"没有。"他诚实地说，"当时就是……本能反应。看到有问题就想解决，看到有漏洞就想补上。可能是职业病吧——我以前在公司的时候也是这样，线上出了事故就什么都顾不上了，一门心思扑上去修。"

姬子笑了。

"这就是答案。"她说，"你不需要定义自己的'位置'。你的本能就是你最好的定位——遇到问题就解决，看到危险就应对。这和战斗力无关，和你'属不属于这里'也无关。"

她端起咖啡杯，像是在敬酒。

"星穹列车不挑人。我们不在乎你从哪来、有多强。我们只在乎一件事——你愿不愿意和我们一起走下去。"

陆辰看着她。

那一瞬间他想起了很多事情——凌晨三点的工位、永远改不完的 bug、吃了半年的外卖、从未用过的年假。

然后他想起了今天——逃命时三月七拉起他的手、丹恒挡在他面前的背影、星对他绽放的笑容、姬子此刻递过来的这杯咖啡。

"你们这个项目组，"他的嘴角忍不住翘了起来，"还缺人吗？"

"你觉得呢？"

"我觉得……你们缺一个坐镇后方看 dashboard 的运维人员。"

"那就这么定了。"

姬子伸出手。

陆辰握住了那只手。

"欢迎加入星穹列车。"

---

一旁默默观察的瓦尔特推了推眼镜。

这位从"另一个世界"来的前辈，从始至终都没有多说什么。但他的眼神在陆辰说出"我不是这个世界的人"时，明显动了一下。

后来，在其他人都各自忙碌的时候，瓦尔特单独找到了陆辰。

"你说你不是这个世界的人。"瓦尔特的声音平和而沉稳，"你对这种经历……害怕吗？"

陆辰犹豫了一下，还是点了点头："说不害怕是假的。"

"那就对了。"瓦尔特微微一笑，"害怕说明你还清醒。真正可怕的不是穿越到陌生的世界——而是在陌生的世界里忘记了自己是谁。"

他拍了拍陆辰的肩膀。

"记住你自己的名字、你的来处、你的信念。无论到了哪里，这些东西都是你的锚。"

陆辰看着这位温文尔雅的前辈，心里涌起一股暖意。

他在游戏里就知道瓦尔特是一个来自"另一个世界"（崩坏3的世界）的人。但在游戏里，这只是一个设定而已。而现在——他面前站着一个和自己一样，经历过"穿越"之苦的前辈。

"瓦尔特先生，"陆辰想了想，"你到这个世界……多久了？"

"很久了。久到我有时候会分不清哪些记忆是'那边'的，哪些是'这边'的。"瓦尔特的语气平淡，但陆辰听出了深处的感慨，"但有一件事我始终记得——我在'那边'的名字，和我在'那边'做过的事。"

"这些就够了吗？"

"这些就够了。"瓦尔特推了推眼镜，"其他的一切——力量也好、命途也罢——都是后来的事。人活着，首先要知道自己是谁。"

陆辰默默点头。

瓦尔特转身要走，忽然又停下来："对了——你的那种能力，看到'底层信息'的能力。"

"怎么了？"

"小心使用。"瓦尔特的声音低了几分，"这个宇宙的'底层'，可能比你想象的要深得多。看到越多，意味着你卷入的东西越深。而有些东西——不是你现在能承受的。"

说完，他真的走了。

陆辰站在原地，咀嚼着这番话。

路易十六看断头台——知道得越多，脖子越凉。

他忽然觉得，这个前辈的话里有一层他暂时无法理解的含义。

## 第十章 Hello World

离开黑塔空间站的那天，空间站的修复工作已经完成了大半。

陆辰站在星穹列车的舷窗前，看着黑塔空间站在星海中渐渐缩小为一个光点。这座巨大的空间站在反物质军团的攻击下伤痕累累，但依然矗立在宇宙中——就像一个被 DDoS 攻击过的服务器，虽然挂了几台机器，但核心服务从未中断。

黑塔的人偶就不来送行了——她在通讯器里冷冷地说了一句"别忘了你答应我的事"就切断了连接。但陆辰注意到，在他离开空间站前，他的那个通讯装置里多了一份文件：

《黑塔空间站技术文档全集（精简版）——编译者专属》

他差点笑出声。黑塔嘴上不说，但该给的资料一份不少。典型的口嫌体正直型甲方。

星穹列车在星海中加速前进，目标是下一个需要"开拓"的世界。

陆辰在列车上被分配了一个小房间——比他在深圳租的那个隔断间大不了多少，但胜在安静、干净，而且窗外的风景是无价的银河星辰。

他坐在床边，掏出手机。手机还有电，但已经完全变成了一个离线设备——没有信号、没有网络、没有那些曾经塞满他生活的 App。

屏幕上还停留着他穿越前最后看到的 IDE 界面。那行报错信息依然醒目：

```
Error: undefined is not a function
```

"undefined is not a function。"陆辰念出声来，忽然觉得这句话有了全新的含义。

在编程世界里，这意味着"你调用了一个未定义的函数"。

而在他现在的处境里——

他自己就是那个"未定义的函数"。

他不属于这个宇宙的任何既有"定义"。没有命途、没有阵营、没有来历。他是一个 undefined——一个未定义的变量被丢进了一个正在运行的庞大系统中。

但他也知道——在编程中，undefined 并不等于 null。Undefined 是"尚未赋值"，而 null 是"空"。

他不是空的。他有自己的经验、知识、记忆和信念。他只是还没有在这个世界中找到自己的"定义"。

"好吧。"他把手机放到一边，站起来走到舷窗前。

列车正在穿越一片色彩斑斓的星云。紫色、蓝色、金色的光芒在舷窗上流淌，像一幅动态的油画。

陆辰看着这片在他的世界中永远不可能亲眼目睹的景色，深深地吸了一口气。

"那就从头来过吧。"

他的双眼中，"编译者之眼"微微闪烁——像一个刚刚完成初始化的系统，正在等待第一条命令。

而在他的视野最边缘、最模糊的地方，那行在黑塔空间站数据库深处瞥见的代码再次浮现了一瞬：

```
// [已注释] function trailblaze_origin() {
```

开拓的起源。

一个被注释掉的、等待被重新激活的函数。

陆辰不知道它意味着什么。但他有一种预感——在未来的旅途中，他会再次看到它。而到了那个时候，他或许已经有能力理解它的含义。

列车窗外，星海无垠。

"Hello World。"陆辰对着宇宙说出了每个程序员学到的第一句话。

你好，新世界。

虽然你的 bug 挺多的，但我来了。
