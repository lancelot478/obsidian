## 视频目标与动机

- **目标：** 将字符级语言模型从简单的二元模型（Bigram）升级为**多层感知机（MLP）**架构
- **动机：** 二元模型只能利用前一个字符来预测下一个字符，建模能力极度受限。本视频实现 Bengio 等人 2003 年发表的经典论文中的神经网络语言模型架构，通过引入"上下文窗口"和"嵌入空间"，让模型基于更长的历史信息进行预测

---

## 核心概念与知识点

### 上下文窗口（Context Window）
- 使用前 $n$ 个字符来预测第 $n+1$ 个字符
- 例如窗口大小为 3 时，单词 "emma" 会被拆解为：`... -> e`, `..e -> m`, `.em -> m`, `emm -> a`

### 嵌入层（Embedding Layer）
- 将离散的字符映射到连续的低维向量空间中
- 含义相近的字符在空间中距离更近

### 特征拼接（Feature Concatenation）
- 将上下文窗口中每个字符的嵌入向量连接起来，作为神经网络的输入

### 非线性激活
- 使用 `tanh` 函数引入非线性，使模型能够学习复杂的模式

### 交叉熵损失（Cross-Entropy Loss）
- 衡量预测概率分布与真实标签之间的差异

## 代码实现步骤

### 1. 数据预处理
- 构建输入 $X$ 和标签 $Y$
- 按上下文窗口大小拆分训练样本

### 2. MLP 架构构建

**参数初始化：**
- 嵌入矩阵 $C$（维度：字符表大小 × 嵌入维度）
- 隐藏层权重 $W_1$、偏置 $b_1$
- 输出层权重 $W_2$、偏置 $b_2$

**前向传播逻辑：**
1. **Lookup：** 通过 `C[X]` 获取上下文字符的嵌入向量
2. **Reshape：** 使用 `.view()` 或 `torch.cat` 将多个字符的向量拼成一个长向量
3. **Hidden Layer：** 计算 `h = torch.tanh(emb @ W1 + b1)`
4. **Logits：** 计算 `logits = h @ W2 + b2`

### 3. 训练循环
- **Mini-batch 训练：** 每次随机抽取一小部分样本（如 32 个）计算梯度并更新
- **学习率寻找：** 通过对数空间搜索找到最优学习率
- **学习率衰减：** 训练过程中逐步降低学习率

### 4. 数据集划分
- **训练集（Training Set）**
- **验证集（Dev/Validation Set）**
- **测试集（Test Set）**
- 用于监控过拟合

## 关键公式

| 公式 | 说明 |
|------|------|
| $z = x \cdot W + b$ | 线性变换 |
| $\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$ | 双曲正切激活函数 |
| `F.cross_entropy(logits, targets)` | 交叉熵损失（数值上比手动计算 log 更稳定） |

## 编程技巧与注意事项

- **高效拼接：** `view()` 操作不涉及内存拷贝，仅改变元数据，因此比 `torch.cat` 极其高效
- **维度陷阱：** 进行矩阵乘法时需非常小心张量的形状（Shape）
- **欠拟合与过拟合：** 隐藏层神经元太少或嵌入维度太低会欠拟合；通过观察验证集损失判断是否过拟合
- **初始化问题：** 权重初始化太乱会导致初始损失非常高，为下一集讨论初始化和 BatchNorm 埋下伏笔
