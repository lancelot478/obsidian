## 核心概念与知识点

### 导数（Derivatives）
- 从导数的数学定义出发，解释当改变一个输入时，函数输出的变化率
- 导数定义：$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$

### 链式法则（Chain Rule）
- 反向传播的核心
- 通过将局部导数相乘，从输出端一层层地计算出对每个输入参数的梯度
- 若 $y = f(u)$ 且 $u = g(x)$，则 $\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$

### 计算图（Computational Graph）
- 将复杂的数学表达式表示为由节点（操作数）和边（操作）组成的**有向无环图（DAG）**

### 反向传播（Backpropagation）
- 在计算图上从后向前应用链式法则，自动计算梯度的过程

### 梯度（Gradients）
- 梯度告诉我们每个权重和偏置应该如何微调，以减小损失函数的值

## 代码实现步骤

### 1. 定义 Value 类
- 创建一个能存储**标量值**及其**梯度**的对象
- 重载 Python 的算术运算符（`__add__`、`__mul__` 等），使 Value 对象之间可以进行加法和乘法
- 在每个运算中保存"前驱节点"（即输入）和执行的操作类型

### 2. 实现局部求导逻辑
- 为每个算术运算编写一个闭包函数（通常命名为 `_backward`）
- 定义该运算如何根据输出梯度计算输入梯度

### 3. 构建自动求导引擎
- 使用**拓扑排序（Topological Sort）**来确保在计算某个节点的梯度之前，该节点之后的所有节点的梯度都已计算完毕
- 实现全局的 `.backward()` 函数，调用拓扑排序并依次执行所有节点的 `_backward`

### 4. 构建神经元模型
- **Neuron 类：** 包含权重 $w$、偏置 $b$ 和激活函数（如 `tanh`）
- **Layer 类：** 一组神经元
- **MLP（多层感知机）类：** 堆叠多个层

### 5. 训练循环（Training Loop）
- **前向传播（Forward Pass）：** 将数据输入模型，计算预测值
- **计算损失（Loss Function）：** 如均方误差（MSE）
- **反向传播（Backward Pass）：** 调用损失函数的 `.backward()`
- **更新权重（Update）：** 根据梯度方向微调参数（简单的梯度下降）

## 关键公式

| 公式 | 说明 |
|------|------|
| $f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$ | 导数定义 |
| $\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$ | 链式法则 |
| $z = \sum(w_i \cdot x_i) + b$ | 神经元线性组合 |
| $a = \tanh(z)$ | 激活函数 |
| $w_{new} = w_{old} - \text{lr} \cdot \frac{dL}{dw}$ | 梯度下降更新规则 |

## 编程技巧与注意事项

- **梯度累加与清零：** 在进行下一次反向传播之前，必须将所有参数的梯度清零（`grad = 0`），否则梯度会累加导致结果错误
- **可视化：** 使用 `graphviz` 库动态可视化计算图，直观理解梯度如何在复杂的表达式中流动
- **数值稳定性：** 虽然 micrograd 专注于标量，但理解张量运算（Vectorization）在实际应用中的高效性也很重要
